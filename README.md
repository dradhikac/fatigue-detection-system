# 💻 Employee Fatigue Detection System

An AI-based fatigue detection system designed to monitor **fatigue levels in software employees** during long working hours. The system uses computer vision and machine learning techniques to analyze facial features such as eye closure and blinking patterns to detect signs of fatigue.

This project integrates **Python for computer vision**, **R for data analysis**, and a **Java-based GUI application** to provide a complete fatigue monitoring solution for workplace environments.

---

## 📌 Features

* 👁️ Real-time face and eye detection using webcam
* 🧠 Machine learning–based fatigue detection
* 💻 Designed for monitoring fatigue in software employees
* 📊 Data analysis and visualization using R
* 🖥️ Java desktop GUI for system interaction
* ⚡ Real-time monitoring during long working hours
* 🗂️ Logging fatigue events for further analysis

## ⚙️ How It Works

1. The system captures video input from the employee’s webcam.
2. Python uses **OpenCV** to detect the face and eye regions.
3. Eye blink patterns and eye closure duration are analyzed.
4. Machine learning models determine fatigue levels.
5. R scripts analyze fatigue data and generate insights.
6. If fatigue is detected, the system can trigger alerts or log the event.
